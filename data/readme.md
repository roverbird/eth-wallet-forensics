## Data Availability

The underlying dataset (Uniswap v3 ETH/USDC swap logs and processed data with ETH addresses metrics) used in the associated research and generated by this open-source software is archived on **Zenodo** at, DOI 10.5281/zenodo.18674643, [Ethereum Wallet Profiling Data: Raw Uniswap Transaction Logs and NBD-Processed Behavioral Features](https://zenodo.org/records/18674644)
